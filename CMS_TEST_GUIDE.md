## 測試 1: 分類功能測試

### 1.1 查看分類列表
1. 訪問 http://localhost:3000/cms
2. 點擊「分類篩選」下拉選單
3. **驗證**：應顯示 4 個分類

### 1.2 分類篩選測試
1. 選擇「系友動態」分類
2. **驗證**：只顯示相關文章
3. 選擇「全部分類」
4. **驗證**：顯示所有文章

### 1.3 分類標籤顯示
1. 查看文章卡片
2. **驗證**：分類標籤正確顯示並帶顏色

3. 點擊「預覽」按鈕
4. **驗證**：預覽對話框正確顯示所有內容

### 2.2 編輯文章時的預覽
1. 編輯現有文章
2. 修改內容
3. 點擊「預覽」
4. **驗證**：預覽顯示最新修改
