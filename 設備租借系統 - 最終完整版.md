# 設備租借系統 - 最終完整版

## 🎉 專案概述

這是一個功能完整的企業級設備租借管理系統，包含前台使用者介面、後台管理功能、使用者認證、進階搜尋等完整功能。

### ✨ 主要特色

- **完整功能**：設備管理、租借申請、審核流程、行事曆檢視
- **CMYK主題**：專業配色設計，每個功能區域都有獨特的視覺識別
- **響應式設計**：完美支援桌面與行動裝置
- **企業級**：適合公司內部使用，包含權限管理
- **優化功能**：本地檔案上傳、進階搜尋、錯誤處理

## 🚀 快速開始

### 環境需求
- Node.js 18.0.0 或更高版本
- npm 或 pnpm 包管理器

### 安裝步驟

1. **解壓專案**
```bash
tar -xzf equipment-rental-system-final-fixed.tar.gz
cd equipment-rental-system
```

2. **安裝依賴**
```bash
npm install
# 或使用 pnpm
pnpm install
```

3. **啟動開發伺服器**
```bash
npm run dev
# 或
pnpm dev
```

4. **開啟瀏覽器**
```
http://localhost:5173
```

5. **建置生產版本**
```bash
npm run build
# 或
pnpm build
```

## 🆘 常見問題

### Q: 如何重設系統資料？
A: 清除瀏覽器的 localStorage 即可重設所有資料。

### Q: 如何新增管理員帳號？
A: 修改 `src/lib/authData.js` 中的 `users` 陣列。

### Q: 如何自訂主題顏色？
A: 修改 `src/App.css` 中的 CSS 變數。

### Q: 如何整合真實後端？
A: 替換 `src/lib/` 目錄下的資料管理模組為 API 呼叫。
