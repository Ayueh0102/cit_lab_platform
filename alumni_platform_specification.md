## 1. 專案概述

本文件旨在闡述「色彩與照明科技研究所系友會社群平台」的系統需求、功能設計、技術架構與使用者體驗目標。此平台旨在建立一個專屬於系友的線上社群，促進系友間的專業交流、職涯發展與情感聯繫，並作為系友會發布官方資訊的管道。

平台的核心理念是**連結**、**分享**與**成長**。我們將透過現代化的網頁技術與人性化的介面設計，打造一個功能強大且易於使用的數位空間。

## 2. 整體架構與設計理念

### 2.1. 技術棧選擇

為了確保平台的穩定性、可擴展性與未來維護的便利性，我們建議採用以下技術棧：

| 層級 | 技術 | 理由 |
| :--- | :--- | :--- |
| **前端** | React.js | 成熟的生態系、元件化的開發模式，能高效建構複雜的互動式介面。 |
| **後端** | Flask (Python) | 語法簡潔、開發快速，且擁有強大的資料處理與機器學習整合潛力。 |
| **資料庫** | PostgreSQL | 功能強大、穩定可靠的開源關聯式資料庫，能處理複雜的資料關係。 |
| **部署** | Docker | 容器化技術能確保開發、測試與生產環境的一致性，簡化部署流程。 |

### 2.2. 主視覺設計：優化的RGB概念

呼應「色彩與照明科技」的專業背景，平台主視覺將以**RGB色彩模型**為靈感來源，但會避免使用高飽和度的純色造成視覺疲勞。設計方向如下：

- **深色模式優先**：採用深灰或近黑色的背景，更能突顯光影與色彩的效果，也符合當代設計趨勢。
- **霓虹光暈效果**：將RGB三色（或其衍生色）作為按鈕、連結、通知等互動元素的光暈或邊框色彩，營造科技感與精緻感。
- **漸層與疊加**：在區塊背景、圖表或裝飾性元素上，使用低透明度的RGB色彩進行漸層或疊加，創造豐富的視覺層次。
- **動態效果**：在滑鼠懸停、頁面載入等狀態加入細微的色彩動態效果，提升使用者體驗的趣味性。

## 3. 核心功能模組

平台將由以下幾個核心功能模組構成：

### 3.1. 系友資料管理系統 (Alumni Profile Management)

此為平台的基礎，提供安全的會員註冊、登入與個人資料管理功能。

- **註冊與登入**：提供Email/密碼註冊，並建議整合 **LinkedIn** 與 **Google** 第三方帳號快速登入，降低使用門檻。
- **個人檔案**：
    - **基本資料**：姓名、畢業年份、班級、聯絡Email、個人網站等。
    - **職涯資料**：目前任職公司、職稱、產業領域、工作經歷、專業技能。
    - **LinkedIn資料導入**：提供「一鍵同步」功能，允許使用者授權後，自動從其LinkedIn帳號匯入並更新職涯資料，大幅提升資料豐富度與即時性。
    - **隱私設定**：使用者可自行設定哪些資料（如Email、電話）僅在特定條件下（如同意交流後）對其他系友可見。

### 3.2. 職缺分享與交流系統 (Job Board & Networking)

此模組旨在促進系友間的互助與職涯發展，而非取代傳統求職平台。核心在於「人脈連結」。

- **職缺發布**：系友可分享內部推薦或已知的職缺資訊，包含公司、職稱、工作描述、地點、薪資範圍（選填）等欄位。
- **交流請求機制**：
    1.  對某職缺感興趣的系友A，可對發布者系友B點擊「**請求交流**」。
    2.  系友B會在其個人後台及Email收到通知，提示系友A希望就此職缺進行交流，並可預覽A的**精簡個人檔案**（如畢業年份、現任公司與職稱）。
    3.  系友B可選擇「**同意**」或「**婉拒**」。
    4.  若選擇「同意」，系統將自動為雙方開啟一個**平台內的私訊對話視窗**，並交換雙方的聯絡Email，方便後續深入溝通。此設計兼顧了隱私與便利性。

### 3.3. 活動發布與報名系統 (Event Management)

作為系友會舉辦線上或線下活動的官方平台。

- **活動建立**：管理員或授權的系友可建立活動，包含活動主題、時間、地點（或線上會議連結）、內容介紹、名額限制、報名截止日期等。
- **活動列表與日曆**：提供列表與月曆兩種檢視模式，讓系友能快速瀏覽近期活動。
- **線上報名**：系友可直接在平台上點擊報名，系統會自動記錄並管理報名名單。
- **活動通知**：在活動前自動發送提醒通知給所有已報名者。

### 3.4. 公佈欄系統 (Bulletin Board)

作為系友會官方資訊發布的管道。

- **公告類型**：可分類為「系友會公告」、「系友動態」、「學術新知」等。
- **發布與管理**：僅限系友會管理員發布與編輯，確保資訊的權威性。
- **置頂功能**：重要公告可被置頂，確保所有登入的系友都能第一時間看到。

## 4. 建議擴充功能

為了讓平台更具吸引力與實用性，建議在核心功能基礎上，規劃以下擴充功能：

- **導師計畫 (Mentorship Program)**：建立一個配對系統，讓資深系友能擔任職場導師，為年輕系友提供職涯諮詢與指導。
- **興趣小組 (Interest Groups)**：允許系友根據地理位置（如：新竹區）、專業領域（如：光學設計）或興趣（如：攝影）自發性成立線上小組，進行更深入的討論。
- **系友地圖 (Alumni Map)**：在地圖上視覺化呈現全球系友的分布（可選擇匿名或顯示），有助於尋找同地區的系友，籌辦地方性聚會。
- **平台內私訊系統 (Direct Messaging)**：除了因職缺交流觸發的對話，也允許任何兩位系友在雙方同意下開啟私訊，進行一對一的交流。
