## 1. 設計哲學：光與數位社群的交會

我們的設計核心是將「色彩與照明」的專業意象，轉化為具體、可互動的數位體驗。我們不直接使用高飽和度的純RGB三原色，而是將其概念昇華為「光暈」、「漸層」與「互動反饋」，營造一個既專業、具科技感，又不失溫度的社群空間。

設計將遵循以下原則：

- **清晰導向 (Clarity-Driven)**：介面佈局與資訊架構將以清晰易懂為最高原則，讓使用者能毫不費力地找到所需功能。
- **體驗一致 (Consistent Experience)**：無論是網頁版還是行動裝置，都將提供一致且流暢的操作體驗。
- **情感連結 (Emotional Connection)**：透過細膩的動態效果與視覺反饋，讓使用者在互動中感受到平台的生命力。

## 2. 視覺系統 (Visual Identity System)

### 2.1. 色彩計畫：沉浸式深色主題與霓虹光暈

為了突顯「光」的主題，我們將採用**深色模式 (Dark Mode)** 作為平台的主要外觀。這不僅能讓色彩點綴更加突出，也符合當代設計趨勢並減少長時間使用的視覺疲勞。

| 色彩用途 | 色碼 (Hex) | 視覺描述 |
| :--- | :--- | :--- |
| **主要背景 (Primary BG)** | `#121212` | 近乎純黑的深灰色，營造沉浸感與專業感。 |
| **區塊背景 (Surface BG)** | `#1E1E1E` | 比主背景稍亮的灰色，用於卡片、彈出視窗等元素，創造層次感。 |
| **主要文字 (Primary Text)** | `#EAEAEA` | 非純白的淺灰色，在深色背景上閱讀舒適不刺眼。 |
| **次要文字 (Secondary Text)** | `#888888` | 用於輔助說明、時間戳等非關鍵資訊。 |
| **邊框/分隔線 (Borders)** | `#333333` | 低調地劃分介面區塊。 |
| **點綴色 - 紅 (Accent Red)** | `#FF4757` | 用於通知、錯誤提示或關鍵操作，帶有霓虹感的亮紅色。 |
| **點綴色 - 綠 (Accent Green)**| `#34E79A` | 用於成功狀態、線上指示燈等，充滿活力的科技綠。 |
| **點綴色 - 藍 (Accent Blue)** | `#4C7FFF` | 用於連結、主要按鈕與選中狀態，穩定而具信賴感的科技藍。 |

### 2.2. 字體排印 (Typography)

我們將選用現代、清晰的無襯線字體，以確保在各種螢幕尺寸上的可讀性。

- **主要字體**: **Inter**。這是一款專為螢幕顯示設計的字體，字重變化豐富，可讀性極佳。
- **字體層級**:
    - **標題一 (H1)**: 32px, Bold
    - **標題二 (H2)**: 24px, Bold
    - **內文 (Body)**: 16px, Regular
    - **按鈕/標籤 (Button/Label)**: 14px, Medium

### 2.3. 圖示 (Iconography)

採用簡潔、一致的**線性圖示 (Line Icons)**，風格與整體科技感保持一致。圖示在懸停或選中時，會顯示對應的霓虹光暈效果。

## 3. 關鍵頁面線框圖與使用者流程

### 3.1. 首頁 / 儀表板 (Dashboard)

使用者登入後的第一個畫面，旨在提供個人化的資訊概覽。

- **佈局**: 採用三欄式佈局。
    - **左欄**: 主導航欄，包含首頁、職缺分享、活動列表、系友名錄、公佈欄等連結。
    - **中欄**: 主要內容區，以卡片形式顯示**最新的公告**與**系友動態**。
    - **右欄**: 輔助資訊區，顯示**即將到來的活動**與**我的待辦事項**（例如：待回覆的交流請求）。

### 3.2. 職缺分享頁 (Job Board)

- **流程**: 
    1. 使用者進入頁面，看到以卡片列表呈現的職缺。
    2. 每張卡片包含職稱、公司、地點、發布者頭像與姓名。
    3. 點擊卡片可展開查看詳細工作描述。
    4. 卡片右下角有「**請求交流**」按鈕。
    5. 點擊後，系統彈出確認視窗，告知對方將會收到通知。
    6. 確認送出後，按鈕狀態變為「已請求」。

### 3.3. 交流請求流程

- **發布者視角**:
    1. 導航欄的「通知」圖示出現紅點（紅色光暈）。
    2. 點擊通知，或進入個人後台，可看到來自系友A的交流請求。
    3. 請求卡片上顯示系友A的簡要資訊（姓名、畢業年份、現職）。
    4. 提供「同意交流」與「婉拒」兩個按鈕。
    5. 點擊「同意交流」，系統自動為雙方開啟私訊視窗，並發送一條系統訊息：「[系友B] 已同意您的交流請求，你們現在可以開始對話了。雙方的聯絡 Email 也已交換。」

## 4. 互動與動態效果：RGB 光暈的應用

- **懸停效果 (Hover)**: 當滑鼠懸停在按鈕、連結或可點擊的卡片上時，物件會出現對應點綴色的柔和光暈（例如，主要操作按鈕為藍色光暈）。
- **載入動畫 (Loading)**: 頁面或組件載入時，會顯示一個由RGB三色光點追逐旋轉的載入指示器。
- **通知提示 (Notification)**: 新通知會以紅色光點或光暈的形式出現在導航欄的「通知」圖示上，直觀且醒目。
